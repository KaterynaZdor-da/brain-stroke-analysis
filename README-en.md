# 🧠 Brain Stroke Data Analysis & Analytics

## 📌 Overview
This project focuses on a comprehensive analysis of health indicators to understand the primary risk factors contributing to brain strokes. The project demonstrates an end-to-end data analytics workflow: extracting data and engineering new features via **Google BigQuery (SQL)**, performing data cleaning and exploratory data analysis (EDA) inside **NotebookLM** / **Jupyter Notebook**, and creating an interactive dashboard in **Tableau**.

## 📊 Data Source & Pipeline

* **Original Dataset:** [Kaggle Brain Stroke Dataset](https://www.kaggle.com/datasets/jillanisofttech/brain-stroke-dataset)
* **Data Selection & Feature Engineering (Google BigQuery):** Raw data was queried to select only the required columns. New categorical features (age groups, glucose status, BMI categories) were engineered using SQL logic.
* **Data Cleaning & Analysis:** The processed data was imported into **NotebookLM** and a Jupyter Notebook, where data cleaning, processing, and Exploratory Data Analysis (EDA) were fully conducted.
* **Visualization:** Tableau Public.
