# 🧠 Brain Stroke Data Analysis & Analytics (Аналіз даних та факторів ризику інсульту)

## 📌 Огляд проєкту
Цек проєкт присвячений комплексному аналізу медичних показників пацієнтів з метою виявлення та розуміння ключових факторів ризику, що призводять до інсульту. У проєкту продемонстровано повний цикл роботи з даними (**End-to-End Data Analytics Workflow**): вибірка даних та створення нових ознак за допомогою **Google BigQuery (SQL)**, безпосереднє очищення, обробка та розвідувальний аналіз (EDA) у **NotebookLM** / **Jupyter Notebook**, а також фінальна візуалізація в **Tableau**.

## 📊 Джерела даних та інструменти

* **Оригінальний датасет:** [Kaggle Brain Stroke Dataset](https://www.kaggle.com/datasets/jillanisofttech/brain-stroke-dataset)
* **Вибірка даних та інженерія ознак (Google BigQuery):** З хмари було відібрано лише необхідні для аналізу стовпці, а також за допомогою SQL-логіки згенеровано нові категоріальні ознаки (вікові групи, статус рівня глюкози, категорії ІМТ).
* **Очистка та обробка даних:** Отриманий набір даних було імпортовано в **NotebookLM** та ноутбук, де відбувався основний етап перевірки, очищення даних та вивчення взаємозв'язків між метриками.
* **Візуалізація:** Tableau Public.
